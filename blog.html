Blender for hand drawn animation?

Last updated Sep 2021

Why?

As 3D tools grow, they also become flexible enough to be suitable for 2D, and even hand drawn animation!

Common workflow expectations

So far in my experiments the GP add-on which maps the mouse wheel to pan the view and scroll the timeline works well. Tools switching is a bit clumsy with many button pressed and could use some tweaking.

Blenders "App Templates" allows you to redesign the whole UI and shortcuts, like the common functionality of saving different views for different tasks, but this adds on that to change the whole workflow if you want to.

What method is managable?

Using D shapes as multi colored 2D gradients, there has been some research papers written on this. The idea being, devices are already specialized in rendering 3D.

Using lights to shape gradients, an other more of a round about way of doing it is lighting an N-gon with spotlight of different colored light.
