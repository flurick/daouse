Strange 3D experiments Tool Prototypes
The idea for this one was to create the a tradigital drawing program with the least amount of ui to still be productive. Brush size, 5 layers per frame, a clear frame button and a slider as a timeline. An interesting puzzle I thought and still am not sure how to work aorund it, is shading 2D shapes. Featherd borders and mesh gradients ar possible answers. I spent some time one summer building a basic shape shading tool in two attempts, one using textured lines (which turned out to overlap itself on wide and bent enough areas, and secondly a layered polygon onto of itself with an offset (this is how blenders greace pencil effect for rim light also works as of writing this text)
